
# Meeting Scheduler

## 📌 Giới thiệu
Dự án quản lý đặt lịch họp thông minh (Spring Boot + ReactJS + MySQL).

## 🚀 Công nghệ sử dụng
- Backend: Spring Boot (Java, Maven)
- Frontend: ReactJS (Vite/CRA)
- Database: MySQL
- Docker (tùy chọn)
